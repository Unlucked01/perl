Система проверки правописания

Добро пожаловать в систему проверки правописания на Perl!

Эта система позволяет проверять орфографию в текстовых документах, предлагает исправления для слов с ошибками и обучается новым словам.

  • Проверка правописания
  • Управление словарем

Возможности системы:

  • Проверка орфографии в текстовых документах
  • Интерактивное исправление ошибок
  • Загрузка файлов для проверки
  • Управление словарем (добавление, удаление, редактирование слов)
  • Обучение системы новым словам
