О проекте "ТВ Программа"

Проект "ТВ Программа" создан для удобного просмотра телевизионной программы передач. Наш сервис позволяет пользователям быстро находить интересующие их передачи, фильтровать контент по категориям и времени показа.

Основные возможности проекта:

  • Просмотр полного списка телеканалов
  • Отображение программы передач на выбранную дату
  • Фильтрация передач по категориям
  • Фильтрация по времени показа
  • Поиск передач по названию

Проект разработан с использованием технологий Perl CGI для серверной части и HTML/CSS/JavaScript для клиентской части. Данные хранятся в базе данных с использованием DB_File.
